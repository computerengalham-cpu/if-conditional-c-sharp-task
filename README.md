Task 1 – Positive, Negative, or Zero (Very Easy)
Write a program that asks the user to enter a number. Using if statements: If the number is greater
than 0 print 'Positive'. If the number is less than 0 print 'Negative'. If the number equals 0 print
'Zero'.
Task 2 – Even or Odd (Easy)
Write a program that asks the user to enter an integer. If the number is even print 'Even Number'. If
the number is odd print 'Odd Number'. Hint: use the modulus operator (%).
Task 3 – Student Grade System (Medium)
Write a program that asks the user to enter a student score from 0 to 100. Use if statements to print
the grade according to the ranges: 90–100 Excellent, 75–89 Very Good, 60–74 Good, 50–59 Pass,
Below 50 Fail.
Task 4 – Simple Login System (Hard)
Create a simple login program. The correct username is 'admin' and the correct password is '1234'.
Ask the user to enter both values. If both are correct print 'Login Successful'. Otherwise print 'Invalid
Username or Password'.
Task 5 – Simple ATM System (Very Hard)
Create a simple ATM withdrawal program. The initial balance is 1000. Ask the user to enter a
withdrawal amount. If the amount is less than or equal to 0 print 'Invalid Amount'. If the amount is
greater than the balance print 'Insufficient Balance'. If the amount is valid and less than or equal to
the balance print 'Withdrawal Successful' and display the remaining balance.
